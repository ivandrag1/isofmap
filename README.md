# AI Кариерен Навигатор (MVP)

Уеб приложение на български за кариерна ориентация в контекста на AI.

## Какво прави приложението
- Показва експозиция на професии към AI на ниво задачи.
- Обяснява защо дадена професия има конкретна оценка.
- Предлага умения за развитие след AI промени.
- Включва въпросник за ученици и препоръчва професионални съвпадения.
- Позволява сравнение на професии и прозрачен преглед на данните.

## Технологии
- React + TypeScript + Vite
- Recharts
- Локални seed данни (без backend)

## Стартиране локално
```bash
npm install
npm run dev
```

## Build за production
```bash
npm run build
```

- Build артефактът е в папка `dist/`.
- `npm run typecheck` е отделна команда за TypeScript проверка.

## Деплой във Vercel
- Framework preset: **Vite**
- Build command: `npm run build`
- Output directory: `dist`
- Препоръчителна Node версия: **20+**

## Архитектура
- `src/data` – seed данни за професии, въпросник и методология.
- `src/utils` – алгоритми за скоринг, обяснения и matching.
- `src/pages` – основни екрани (начало, професии, детайл, въпросник, резултати, сравнение, метод, data view).
- `src/components` – reusable UI и графики.

## Метод накратко
- Task Exposure Score = `0.40*automation + 0.25*augmentation - 0.15*judgment - 0.10*social - 0.10*physical`
- Profession Exposure Score = претеглена средна по важност на задачите.
- Извеждат се още: риск от автоматизация, AI подпомагане и човешко предимство.

## Важно
Резултатите от въпросника и оценките са ориентировъчни. Инструментът подпомага кариерна рефлексия и планиране, но не взема окончателни решения вместо потребителя.
